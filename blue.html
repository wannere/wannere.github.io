Sushi Icons

This project was to create a set of six thematic icons using a limited color palette. I did some homework at a conveyor sushi restaurant and ended up with this icon set.

Woodcut Letters

In this project I needed to apply something we learned in class to a letter system other than Roman letters (what we use in English). I chose to use Japanese Hirigana letters and was inspired by woodcut printing press letters of the late 1800s, and the letters say the word "yes".

Frutiger Type Specimen

This project was to create three text specimen posters showing a typeface that was randomly assigned. We were allowed to use type, shapes, color and lines and each of the three posters had to be aligned to a different grid: straight, diagonal and free space.

Helvetica Type Specimen

This project was similar to the Frutiger type specimen project in that each of three posters needed to be on a straight, diagonal, or free placement grid. This project also required the use of images.
